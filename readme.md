# Python Fundamentals Projects

Collection of Python projects completed while learning core Python programming concepts.

## Objectives

This repository focuses on:

- Functions
- Dictionaries
- Control flow
- Loops
- Error handling
- String manipulation
- Basic software design
- Command Line Interfaces (CLI)

---

## Projects

### 1. Text Analyzer

A command-line application that analyzes a text provided by the user.

#### Features

- Remove punctuation
- Convert text to lowercase
- Count word frequencies
- Find the most frequent word(s)
- Display the top N most frequent words

#### Concepts Practiced

- Dictionaries
- Functions
- String manipulation
- Sorting algorithms
- Error handling

---

### 2. Task Manager CLI

A simple command-line task management application.

#### Features

- Add tasks
- Display tasks
- Mark tasks as completed
- Delete tasks
- Interactive menu system

#### Concepts Practiced

- Dictionaries
- Functions
- User input validation
- CLI design
- State management
